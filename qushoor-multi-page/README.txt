Qushoor Multi-Page Website

Files:
- index.html
- products.html
- about.html
- cart.html
- checkout.html
- login.html
- register.html
- style.css
- script.js
- assets folder

How to run:
1. Open index.html in your browser.
2. Register a new account.
3. Add products to cart.
4. Go to checkout and place order.

Netlify upload:
1. Go to netlify.com
2. Add new site
3. Deploy manually
4. Drag the full qushoor-multi-page folder, not just index.html

Important:
This version uses localStorage, so login/cart/orders are saved on the same browser only. For real online orders saved from all users, connect Firebase/Supabase/backend later.
